Variable: es una forma de almacenar algo para ser usado mas tarde
String: es un objeto que sirve para representar y manipular 
una secuencia de caracteres. Esta tiene propiedades y metodos.
Funciones: Son bloques de construcción similar a un procedimiento,
un conjunto de instrucciones que realiza una tarea o calcula algun 
valor. Para que funcione debe de tener una entrada (argumentos) 
y una salida, (return)
Declaraciones if: es una estructura de control, que se utiliza para
tomar decisiones. Primero evalúa una expresión, si da resultado
positivo, se realizan las acciones relacionadas con el caso positivo
Booleano: es un tipo de dato primitivo, por definición, tiene dos
valores, (true y false)
